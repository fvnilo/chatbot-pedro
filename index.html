This is a simple experiment on building a chatbot.

Desired features

  • Clever replies;
  • Subject and keyword memory;
  • Knowledge about many human subjects such as love, life, relationships, etc.
  • Web-based interaction.

You will need:

  • Python 2.7
  • PyAIML
  • PyYAML

To Run it:

python main.py

You should see file learning statements with time before the prompt.
